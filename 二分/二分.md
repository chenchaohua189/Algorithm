## 使用场景
(1) 找到可以将区间划分为[ 满足条件 | 不满足条件 ] 或 [ 不满足条件 | 满足条件]的数, 这个边界就是答案

## 模板1: 
```c++
int bsearch(int l, int r){
    while (l < r){
        int mid = (l + r) / 2;

        if (check(mid)) l = mid + 1;
        else r = mid;
    }

    return l;
}
```

## 模板2:
```c++
int bsearch(int l, int r){
    while (l < r){
        int mid = (l + r + 1) / 2;

        if (check(mid)) l = mid;
        else r = mid - 1;
    }

    return l;
}
```
